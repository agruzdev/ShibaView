## Installation

Prebuilt ShibaView executable requires MSVC 2026 x64 runtime on Windows, so please install the latest version of [Microsoft Visual C++ Redistributable x64](https://aka.ms/vc14/vc_redist.x64.exe)


## Usage

Use F1 to open menu with controls description


## Building for Linux

For using Qt6 gcc prebuilt binaries:

```
sudo apt-get install libglu1-mesa-dev freeglut3-dev mesa-common-dev mesa-utils libopengl-dev
sudo apt-get install libxcb.*-dev
```
